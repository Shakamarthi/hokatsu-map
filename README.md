# 保活管理システム v2.4

## GitHubへアップロードするファイル

今回更新するのは次の2ファイルです。

- `index.html`
- `hokatsu-data.json`

GitHubリポジトリ `shakamarthi/hokatsu-map` の直下に、同名ファイルとして上書きしてください。

## スマホでの更新手順

1. ZIPをiPhoneの「ファイル」アプリで展開
2. GitHubで `shakamarthi/hokatsu-map` を開く
3. `Add file` → `Upload files`
4. `index.html` と `hokatsu-data.json` を選択
5. `Commit changes`
6. 数分後に以下を開いて確認

`https://shakamarthi.github.io/hokatsu-map/?v=2.4`

## 主な追加機能

- 園一覧から「見学評価を見る」をタップして評価内容を表示
- 総合評価
- 主な強み
- 主な懸念
- 家庭運用との適合性
- 追加確認事項

## GitHub連携について

現在のChatGPT GitHub接続はリポジトリ閲覧はできますが、ファイル更新は403で拒否されます。そのため、現時点ではスマホからの手動アップロードが必要です。
