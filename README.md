# 保活管理システム v2.6.3

## 今回の変更

- 各園の通園情報を「徒歩時間」と「自転車時間」の両方で表示
- トップ画面では `🚶 徒歩○分｜自転車○分` と表示
- 詳細画面でも徒歩・自転車を分離して表示
- 起点住所そのものは、HTML・JSON・README・画面・リンクに保存していません
- 各園には計算結果だけを保持しています
- PWAキャッシュを `hokatsu-map-v263` に更新

## 精度について

通園時間は経路・信号・自転車の走行条件によって変動します。
今回の値は候補比較用の概算値です。実際の送迎判断では、利用する地図アプリで経路を最終確認してください。
