# 保活管理システム v2.6.2

## 今回の修正

- 公式HPボタンをGoogle検索ではなく、確認済みの園公式ページへ直接リンク
- 公式施設ページが確認できない園は「公式HP未登録」と表示
- 保活管理Excel、戦略レポート、応募圧シミュレーション、見学評価を突合
- 通園時間、開始月齢、延長終了時刻、園庭、ICT、用品・おむつ、保育内容を再反映
- 詳細画面に「情報の根拠」を追加
- 推測と確認済み情報を分離し、本当に未確認の項目だけ「要確認」と表示
- PWAキャッシュを `hokatsu-map-v262` へ更新

## 注意

延長保育の終了時刻が確認できても、0歳児が利用できるとは限りません。
0歳児の利用条件が確認できていない園は、終了時刻を表示した上で「0歳要確認」としています。

ICTについては「導入あり」が確認できていても、アプリ名・欠席連絡・登降園打刻などの機能単位で未確認の場合があります。
そのため、詳細画面では確認済み範囲と残課題を分けています。
